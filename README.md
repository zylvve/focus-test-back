# Тестовое задание (бэк)

Для сборки и запуски проекта используйте ```docker-compose up --build```. По умолчанию сервер доступен по ```http://localhost:8000/```. Cоздаются контейнеры для сервера и БД на PostgreSQL, миграция проходит автоматически при запуске.
